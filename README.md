# Stickers Shop App

Welcome to the Stickers Shop App repository! This app allows users to order stickers from the provided catalogue or create their own custom stickers. Users earn 1% of the profit generated from the sales of the stickers they create, excluding their own purchases.

## Features

- Browse and order stickers from the catalogue.
- Create custom stickers with personalized designs.
- Earn 1% of the profit from sales for each sticker created (excluding self-purchases).
- Track earnings and sales statistics in the admin dashboard.
- Manage orders, payments, and shipping information.
- Update and delete stickers from the catalogue (admin access only).
- Rate and review purchased stickers.
- Receive notifications for order updates and promotions (not implemented yet).

## Usage

Upon launching the Stickers Shop App, users can perform the following actions:

- Browse the sticker catalogue and add desired stickers to the cart.
- Create custom stickers by specifying their designs and adding them to the catalogue.
- Place orders for selected stickers, providing the necessary shipping and payment details.
- Access the user dashboard to view earnings, sales statistics, and order history.
- Update or delete stickers in the catalogue (admin access only).
- Rate and review purchased stickers to share feedback with the community.
